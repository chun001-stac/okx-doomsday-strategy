## 📝 步骤1：注册Bybit账户（5分钟）

### 1.1 访问Bybit官网
- 网址：https://www.bybit.com
- 点击右上角**注册**
- 使用手机号或邮箱注册（中国用户可用）

### 1.2 完成基础验证
- 邮箱验证
- 手机验证（可选）
- 设置交易密码

### 1.3 访问模拟交易
1. 登录Bybit账户
2. 右上角点击**模拟交易**（或访问：https://www-testnet.bybit.com）
3. 进入**合约交易**界面

## 🔑 步骤2：创建模拟交易API（5分钟）

### 2.1 进入API管理
1. 在模拟交易界面，点击右上角头像
2. 选择**API管理**
3. 点击**创建API**

### 2.2 设置API参数
- **API名称**：`DoomsdaySimulation`
- **权限设置**（全部勾选）：
  - ✅ 合约交易
  - ✅ 钱包（余额查询）
  - ❌ 提现（模拟交易无此选项）
- **IP限制**：留空（允许所有IP）
- **创建并保存**

### 2.3 获取API信息
保存以下信息（不要泄露）：
```
API Key: [你的API Key]
Secret Key: [你的Secret Key]
```

**注意**：Bybit模拟交易**不需要Passphrase**

## ⚙️ 步骤3：配置交易系统（5分钟）

### 3.1 修改配置文件
打开`/root/.openclaw/workspace/freqtrade_workspace/config.ini`，修改为：

```ini
[Bybit]
# Bybit模拟交易API
api_key = [你的Bybit API Key]
api_secret = [你的Bybit Secret Key]
api_password =  # 留空，Bybit不需要

[Trading]
# Bybit交易对格式
symbol = ETH/USDT:USDT  # ETH永续合约
leverage = 10           # 10倍杠杆
position_size_pct = 0.15  # 15%仓位（模拟测试用较小值）
enable_trading = true   # 启用交易
test_mode = false       # 实盘模式（但使用模拟资金）

[Strategy]
timeframe = 5m
# 其他参数保持不变...

[Risk]
stoploss_pct = 0.25
takeprofit_pct = 0.30
max_daily_loss_pct = 0.30
max_total_loss_pct = 0.50

[System]
enable_trading = true
test_mode = false
check_interval = 60
```

### 3.2 创建Bybit专用交易脚本
已为你创建：`bybit_doomsday_sim.py`

## 📊 步骤5：监控与评估（3-5天）

### 每日检查指标
1. **胜率**：应>35%
2. **盈亏比**：应>1.5
3. **做空占比**：应≈75%
4. **最大回撤**：应<20%

### 关键观察点
- **做空信号是否正常触发**
- **杠杆是否正确应用**
- **止损止盈是否有效**
- **仓位管理是否合理**

## 🔧 步骤6：问题排查

### 常见问题
#### Q1: API连接失败
```
检查：1. API密钥是否正确 2. 是否为模拟交易API 3. 网络连接
```

#### Q2: 无法下单
```
检查：1. 模拟账户余额 2. 交易对格式 3. 权限设置
```

#### Q3: 做空信号不执行
```
检查：1. 策略逻辑 2. 市场数据 3. 信号过滤
```

#### Q4: 杠杆设置失败
```
检查：1. Bybit合约类型 2. 杠杆限制 3. 保证金模式
```

grep -i error bybit_sim.log

# 查看交易记录
grep -i "下单\|平仓" bybit_sim.log

# 查看信号统计
